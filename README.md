# JOHN VENN

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **LOGIKĒ**.

JOHN VENN (1834–1923) — LOGIKĒ, the VISUAL LOGIC. Boole & Jevons made logic a matter of combinations; Venn made those combinations something you can SEE. ⚑ THE VENN DIAGRAM (1880): overlapping circles divide the plane into REGIONS, one per combination of in/out (in A&B not C; in all three; in none) — so a logical statement becomes a PATTERN OF SHADING ('no A are B' = blacken the A∩B lens) and a syllogism becomes a PICTURE: shade what the premises forbid, read the conclusion off what's left. The most successful NOTATION in the history of logic — the one diagram that escaped the seminar room onto every whiteboard. LIVE: a 3-set Venn — toggle categorical premises, watch regions shade empty (red), the survivors & the A–C conclusion read off the picture (verified All A→B + All B→C shows All A→C). CONFIDENCE-TIERED: STRONG (a SOUND, COMPLETE visual method for class logic — every region a real combination, validity literally visible; arguably the most-recognised object in all of math) / EULER-CAME-FIRST (⚠ Leonhard Euler drew logical circles a century earlier; Venn's innovation = always draw ALL regions then SHADE, so the diagram can represent ignorance — he called them improved 'Eulerian circles') / BREAKS-DOWN-PAST-3–4-SETS (gorgeous for 2–3 classes, awkward at 4 (ellipses), unreadable beyond — the eye is not a general computer). ⚑ The 8 regions of a 3-circle diagram ARE the 8 rows of a 3-variable truth table = [[william-stanley-jevons]]'s 8 piano keys = a Karnaugh map; shade-to-eliminate is how circuit engineers still simplify gates. Also a frequency-probabilist; built a cricket bowling-machine that clean-bowled an Australian star. The truth table you can look at.
